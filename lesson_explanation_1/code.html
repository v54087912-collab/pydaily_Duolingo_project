Printing to the Console

In Python, we use the print() function to show text on the screen. It's like telling the computer to say something out loud.

main.py 
# This will print a message
print("Hello, World!")
print("I am learning Python.")
terminal Output
> Hello, World!
> I am learning Python.
info

Did you know?

You can print numbers directly without quotes, like print(42).
